veículos disponíveis na empresa, e o framework utilizado foi o

## Como Rodar?
Abra um terminal em seu sistema (PowerShell, se estiver no Windows)
e rode o seguinte código:

`git clone https://github.com/GlaucoFleuryCM/Web_projeto.git`

A partir daí, caso você ainda não o tenha instalado, instale o 
[npm (Node Packet Manager)](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm).
Com ele instalado, entre no diretório do projeto e rode:

`npm install`

Algumas depêndencias serão instaladas. Para abrir o site no 
seu browser local, basta rodar:

`npm start`
